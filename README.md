# Intro
A simple fish catching 2D game using SpriteKit.

## Getting Started
To get a local copy up and running follow these simple steps.
1. Install Xcode 15.2 or higher

2. Clone the repo
`git clone https://github.com/Thush-Fdo/CatchFishy.git`


#### Built With
- Xcode 15.2
- iOS 13.0+
- SpriteKit
- GameplayKit
